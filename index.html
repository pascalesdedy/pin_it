Pin it

Pinterest clone using ruby on rails

View on GitHub Download .zip Download .tar.gz

This is Pinterest Clone written in Ruby on Rails.

Installation :

  • Run git clone
  • cd to the app folder
  • Run bundle install
  • Run rake db:migrate or bundle exec rake db:migrate
  • Run rails server
  • Open localhost:3000 on your browser

Integrated with devise authentication, so visitor can create account for them.
Bootstrap-sass for styling paperclip gem for uploading image
paperclip for Image Uploading
Masonry for styling
